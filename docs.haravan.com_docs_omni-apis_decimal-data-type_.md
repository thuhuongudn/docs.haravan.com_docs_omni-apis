# Decimal Data Type

`Version: 1.0`

A signed decimal number, which supports arbitrary precision.

- Maximal 22 digits before decimal point.

- Maximal 4 digits after decimal point.


Example values:

`29.99`

`29.999`

`29.9999`

`299999.9999`
